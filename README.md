## 🇬🇧 English
Hello, world!
Thanks to the GitHub team and the global open-source community who make it possible to share this folder.

This repository is a **test sandbox** for learning Git—the distributed version control system created by **Mr. Linus Torvalds** (also the creator of the Linux kernel).

> ⚠️ **This is NOT a tool or a product. This is purely a personal test / learning artifact.**

### Why Open Source?
I chose to publish this test openly because open source:
- 🌍 **Knowledge is shared freely** — anyone, anywhere can learn from it.
- 🤝 **Collaboration knows no borders** — bugs are caught faster, ideas come from everywhere.
- 🔍 **Transparency & trust** — code can be inspected, verified, and improved by the community.
- 📚 **Learning by doing** — beginners can see real workflows, not just textbooks.
- 🚀 **It accelerates progress** — standing on the shoulders of giants.
